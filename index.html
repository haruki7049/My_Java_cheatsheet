Hello Worldを書こう

Mainクラスにmain関数を置いて、println関数書いて、終わり!! 閉廷!!

クラス名はファイル名と同じ名前にするのが慣習らしい。そして、ファイル名の始めを大文字にしないといけない。らしい。 エントリポイントは、main関数と呼ばれる特殊な関数を書く。”public”と”static”、そして引数として"String[] args"を書く。 Javaにおけるprint関数(改行込み)は、System.out.println()。そのまま書けばほとんどの場合実行可能。

ソースコード例

                        
public class Main{
    public static void main(String[] args){
        System.out.println("Hello!!");
    }
}

publicを付けないとどうなるか

publicを付けないと、単純に言うと他のプログラムから見えなくなる。 つまり、classの”public”を消すとMainクラスが見つからなくなり、エントリポイントの”public”も同様。 コンパイルは成功するが、実行が不可能になる。
